# 📜 Changelog

Все значимые изменения проекта.

---

## [1.12] - 2026-04-14

### Added
- Автоматический ежедневный бэкап всех важных файлов
- Кнопка «⏰ Выставить автобэкап» в админ-панели
- Возможность изменять время автобэкапа через бота
- Сохранение настроек автобэкапа в settings.json

### Improved
- Значительно улучшена стабильность системы кэша
- Добавлена гибкая система PROXY_MODE (none / telegram / lk / all)
- Улучшена работа с нестабильными SOCKS5-прокси
- Исправлена ошибка "message is not modified" в админ-панели

### Fixed
- Повторное срабатывание автобэкапа в одну минуту
- Проблемы с редактированием сообщений в админке

## [1.11] - 2026

### Added

* 📦 новая система кэширования страниц (`page_cache.pkl`)
* 🧠 вынесен парсер в отдельный модуль `parser.py`
* 🏗️ базовый класс `ScheduleParser` для кастомных парсеров
* 🔌 фабрика `get_parser()` для удобной замены парсера

### Improved

* ускорена загрузка расписания за счёт кэша
* снижена нагрузка на сайт ПГУТИ
* улучшена архитектура проекта (разделение логики)
* кэш недели (`wk_cache`) теперь сохраняется и восстанавливается

### Fixed

* ❌ дублирование `_load_cache_file` и `_save_cache_file`
* ❌ некорректное сохранение кэша
* ❌ варнинг `Не удалось сохранить кэш-файл: 0`
* ❌ потеря кэша при перезапуске бота
* ❌ проблемы с определением текущей недели после рестарта
