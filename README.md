# 📅 ТГ БОТ КС ПГУТИ — Расписание

Телеграм-бот для просмотра расписания КС ПГУТИ.
Поддерживает группы, рассылки, статистику и админ-функции.

---

## 🚀 Возможности

* 📆 Просмотр расписания (сегодня / неделя)
* 🔁 Переключение недель через кнопки
* 🔔 Автоматические рассылки
* 📦 Умный кэш (ускорение загрузки и снижение нагрузки)
* 👥 Support mode (ответ пользователям)
* 📊 Статистика использования
* 🛠️ Админ-панель
* 🐳 Поддержка Docker

---

## 🆕 Обновления (последний патч)

### 📦 Новый кэш

* Реализован единый кэш страниц (`page_cache.pkl`)
* Добавлен TTL (время жизни записей)
* Кэш сохраняется на диск при завершении работы
* Кэш загружается при старте бота
* Устранён баг с дублирующими функциями `_load/_save_cache`

👉 Результат:

* меньше запросов к сайту
* быстрее отклик бота
* стабильная работа после перезапуска

---

### 🧠 Новый модуль парсера

Парсер полностью вынесен из основного файла в отдельный модуль:

```
parser.py
```

Теперь:

* логика парсинга изолирована
* можно легко менять парсер без трогания бота
* добавлена архитектура через базовый класс

#### Основные функции:

```python
parse_schedule_pretty(html)
get_current_week_from_html(html)
```

#### Поддержка кастомных парсеров:

достаточно реализовать класс-наследник `ScheduleParser`

## ⚠️ Ограничения текущей реализации

Несмотря на введённую модульность (`parser.py`), на данный момент проект **частично остаётся привязанным к ПГУТИ**.

### 📌 Что это значит:

* логика парсинга и форматирования ориентирована на структуру сайта ПГУТИ
* автоопределение текущей недели реализовано под `lk.ks.psuti.ru`
* структура URL (`wk`, `obj` и т.д.) жёстко используется в коде
* часть бизнес-логики (кнопки, навигация) зависит от текущего формата расписания

### ⚙️ Вывод

Модульность уже заложена (через `ScheduleParser`), но:

> 🔧 для полной универсальности требуется дальнейшее разделение логики
> (например: вынести работу с неделями, URL и кнопками в отдельные адаптеры)

## ⚙️ Переменные окружения

Создай `.env` файл:

```env
RELEASE_TOKEN=токен бота (через @BotFather)
OWNERID=твой Telegram ID
PROXY_URL=socks5://user:password@host:port  # (опционально)
```

## 🛠️ Установка

### 🐳 Docker (рекомендуется)

```bash
git clone https://github.com/Sp0nge-bob/TGBOT
cd TGBOT

cp .env.example .env
nano .env

docker compose up -d --build
```

---

### 🐍 Python

```bash
git clone https://github.com/Sp0nge-bob/TGBOT
cd TGBOT

python3 -m venv venv
source venv/bin/activate

pip install -r requirements.txt

cp .env.example .env
nano .env

python tg.py
```

## 📊 Основные команды

### 👤 Пользователь

* `/start` — запуск и выбор группы
* `/today` — расписание на сегодня
* `/week` — расписание на неделю

---

### 🛠️ Админ

* `/admin` — админ-панель
* `/stats` — статистика
* `/list_users` — пользователи
* `/schedule_list` — активные рассылки
* `/broadcast` — массовая рассылка
* `/clear_sent` — сброс отправок
