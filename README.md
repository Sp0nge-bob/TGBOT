# 📅 ТГ БОТ КС ПГУТИ — Расписание

Телеграм-бот для просмотра расписания КС ПГУТИ.
Поддерживает группы, рассылки, статистику и админ-функции.

---

## 🚀 Возможности

* 📆 Просмотр расписания (сегодня / неделя)
* 🔁 Переключение недель через кнопки
* 🔔 Автоматические рассылки
* 📦 Умный кэш (ускорение загрузки и снижение нагрузки)
* 👥 Support mode (ответ пользователям)
* 📊 Статистика использования
* 🛠️ Админ-панель
* 🐳 Поддержка Docker

## ⚙️ Переменные окружения

Создай `.env` файл:

```env
RELEASE_TOKEN=токен бота (через @BotFather)
OWNERID=твой Telegram ID
PROXY_URL=socks5://user:password@host:port  # (опционально)
 === РЕЖИМ ПРОКСИ ===
PROXY_MODE=none
 Возможные значения:
 none      - ничего не проксировать (рекомендуется)
 telegram  - только запросы к Telegram API
 lk        - только запросы к lk.ks.psuti.ru
 all       - проксировать всё
```

## 🌐 Прокси

Бот поддерживает гибкий режим проксирования через переменную `PROXY_MODE`:

* `none` — ничего не проксировать (по умолчанию)
* `telegram` — проксировать только Telegram API
* `lk` — проксировать только запросы к `lk.ks.psuti.ru`
* `all` — проксировать и Telegram API, и запросы к сайту расписания

`PROXY_URL` используется как адрес SOCKS5/HTTP прокси в выбранном режиме.

## 🧩 Модульная структура

Основной файл запуска: `tg.py`, логика вынесена по модулям:

* `bot_modules/core.py` — единый инфраструктурный модуль (config + storage + stats + broadcast)
* `bot_modules/admin_panel.py` / `bot_modules/admin_logic.py` — логика админ-панели и admin-хендлеров
* `bot_modules/user_logic.py` — логика пользовательских хендлеров расписания

Это упрощает сопровождение и независимое тестирование частей бота.
